# One Piece DevOps em Go

Este é um projeto inicial em **Go** com foco em práticas básicas de DevOps.  
Ainda está em construção, mas já possui código principal, testes e configuração de módulo.

---

## 🚀 Como rodar o projeto

1. **Instale o Go** (versão 1.20 ou superior).  
   - [Download Go](https://go.dev/dl/)

2. **Clone o repositório**:
   ```bash
   git clone https://github.com/taiones/onepiece-devops-go.git
   cd onepiece-devops-go
3. **Inicialize o módulo Go**
   ```bash
   go mod tidy
4. **Execute a aplicação**
   ```bash
   go run main.go
   [O servidor deve iniciar e responder no terminal.]
   [Acesse em http://localhost:8080]

## 🧪 Como rodar os testes

1. **No diretório do projeto, execute:**
   ```bash
   go test ./...
3. **O Go vai rodar todos os testes definidos em main_test.go.**
4. **Se tudo estiver certo, você verá mensagens de sucesso no terminal.**
